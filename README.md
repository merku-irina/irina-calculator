# Калькулятор типа внешности по Кибби

AI-калькулятор определяет тип внешности по методу Дэвида Кибби на основе двух фотографий. Пользователям не нужно вводить никаких ключей — всё работает автоматически.

## Структура файлов

```
index.html              — страница
style.css               — стили
app.js                  — логика фронтенда
api/
  analyze.js            — серверная функция Vercel (прокси к Anthropic API)
vercel.json             — конфигурация Vercel
kibbe_types_guide.pdf   — гид по всем типам
```

---

## Публикация на Vercel (пошагово)

### Шаг 1 — Загрузите файлы на GitHub

1. Зайдите на github.com и создайте новый репозиторий (например `kibbe-calculator`)
2. Загрузите все файлы, сохранив структуру папок — важно, чтобы папка `api/` была внутри репозитория

### Шаг 2 — Подключите репозиторий к Vercel

1. Зайдите на vercel.com и войдите через GitHub
2. Нажмите Add New → Project
3. Выберите ваш репозиторий `kibbe-calculator`
4. Настройки оставьте по умолчанию

### Шаг 3 — Добавьте API-ключ (до нажатия Deploy)

На том же экране найдите раздел Environment Variables и добавьте:

  Name:  ANTHROPIC_API_KEY
  Value: sk-ant-...ваш ключ...

Получить ключ: console.anthropic.com → API Keys → Create Key

Теперь нажмите Deploy — через минуту сайт будет доступен.

## Как это работает

  Пользователь → ваш сайт на Vercel
                      ↓
               /api/analyze  ← ключ хранится здесь, на сервере
                      ↓
             Anthropic API

Пользователи открывают страницу и сразу работают — никаких ключей вводить не нужно.
